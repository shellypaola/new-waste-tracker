# Waste Warrior - Design System Documentation
## Final Version 1.0

---

## Executive Summary

Waste Warrior uses a **clean, minimal design system** with a warm gradient background, white cards for clarity, and strategic color accents for alerts and gamification. The system prioritizes **readability, accessibility, and emotional warmth** to make food waste reduction feel inviting rather than clinical.

## â™¿ Accessibility

### Contrast Ratios (WCAG AA)
- âœ… Primary text on white: 11.2:1
- âœ… Secondary text on white: 7.0:1
- âœ… White text on teal: 7.2:1
- âœ… Critical text: 7.1:1
- âœ… Warning text: 5.2:1
- âœ… All buttons: >4.5:1

### Touch Targets
- Minimum: 44x44px (iOS), 48x48px (Android)
- Preferred: 48x48px for all interactive elements
- Spacing: 8px minimum between targets

### Screen Reader Support
- All icons have text labels
- Status is announced ("Critical: Expires today")
- Buttons have clear actions ("Mark as used")
- Time remaining announced ("3 days left")

### Motion Preferences
```
If prefers-reduced-motion:
  - Disable all animations
  - Use instant transitions
  - Show/hide without movement
  - Keep haptic feedback
```

### Color Blindness
- âœ… Never rely on color alone
- âœ… Use icons + text for status
- âœ… Border thickness indicates urgency
- âœ… Pattern fills available for critical modes

## ğŸ“Š Visual Hierarchy

### Priority Levels

**Level 1 - Critical Actions (Red):**
- 4px thick border
- High contrast text (#C62828)
- Solid green CTA button
- Top of "Items at Risk" list

**Level 2 - Warnings (Orange):**
- 3px medium border
- Medium contrast text (#E65100)
- Outlined CTA button
- Middle of "Items at Risk" list

**Level 3 - Normal Items (Gray):**
- 1px thin border
- Standard text (#616161)
- No CTA needed
- Regular inventory list

**Level 4 - Gamification (Teal):**
- Bold gradient background
- White text (high contrast)
- Celebratory, special treatment
- Separated section

### Information Density

**Dashboard (High Priority):**
- Large numbers (32px)
- Minimal text
- Focus on at-risk items
- Quick actions visible

**Inventory (Medium Priority):**
- Standard cards (16px text)
- Category filters prominent
- Search easily accessible
- Scannable list

**Analytics (Low Priority):**
- Detailed charts
- Smaller text acceptable
- More information density
- Deep-dive content
